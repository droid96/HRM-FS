================================================================================
 IMPORTANT NOTES
================================================================================

1. SHADCN/UI COMPONENTS REQUIRED:
   This project uses shadcn/ui components. You must have these in /components/ui/:
   
   - button.tsx
   - dialog.tsx
   - input.tsx
   - label.tsx
   - select.tsx
   - switch.tsx
   - tabs.tsx
   - calendar.tsx
   - badge.tsx
   - textarea.tsx
   - card.tsx
   - (and other standard shadcn components)

   To add shadcn components:
   npx shadcn-ui@latest add button
   npx shadcn-ui@latest add dialog
   npx shadcn-ui@latest add input
   ... (repeat for all components)

2. TAILWIND CSS V4:
   The project uses Tailwind CSS v4 with @theme inline directive.
   Ensure your tailwind.config is properly set up.

3. MOCK DATA:
   All employee data, events, emails, etc. are mock data for demonstration.
   Replace with real API calls in production.

4. LOCALSTORAGE:
   User preferences (dashboard widgets, quick actions) are saved in localStorage.

 DEPLOYMENT INSTRUCTIONS
================================================================================

METHOD 1: Manual Setup
----------------------
1. Create new Vite + React + TypeScript project:
   npm create vite@latest hr-portal -- --template react-ts

2. Install dependencies:
   npm install

3. Install Tailwind CSS v4:
   npm install -D tailwindcss@next @tailwindcss/vite@next

4. Copy all files from this package to your project

5. Install shadcn/ui:
   npx shadcn-ui@latest init
   npx shadcn-ui@latest add button dialog input select switch tabs calendar badge label textarea

6. Run development server:
   npm run dev

================================================================================
 CUSTOMIZATION GUIDE
================================================================================

TO CHANGE COLORS:
Edit /styles/globals.css:
- --primary: #007AFF (main blue)
- --chart-1 through --chart-5 (accent colors)
- Dark mode variables in .dark { }

TO ADD NEW PAGES:
1. Create component in /components/pages/
2. Add to App.tsx switch statement
3. Add menu item to Sidebar.tsx

TO MODIFY QUICK ACTIONS:
Edit ALL_AVAILABLE_ACTIONS in QuickActionsCustomizer.tsx

TO ADD NEW WIDGETS:
1. Create widget component
2. Add to Dashboard.tsx
3. Add toggle in WidgetCustomizer.tsx
