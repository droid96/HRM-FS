==============================================
KEY FEATURES
==============================================

✅ iOS-inspired design with San Francisco fonts
✅ Fully functional dark mode toggle
✅ Customizable dashboard widgets
✅ Quick Actions customizer (select your favorite 6 actions)
✅ Employee onboarding with automated email workflows
✅ Employee deboarding with multi-level approval (HR → Finance → Manager)
✅ Interactive calendar with events
✅ HR AI Assistant chatbot with employee database access
✅ Context-aware Email page (shows onboarding emails for HR, approvals for managers)
✅ Document generator with auto-fill from employee database
✅ Attendance tracking with manual check-in
✅ Payroll management
✅ User management and access control in Settings
✅ Blue and purple gradients throughout
✅ Glassmorphism effects
✅ Light outlines on all tiles
✅ Glossy icons with gradient overlays
